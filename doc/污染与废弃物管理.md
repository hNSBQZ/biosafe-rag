# 执行概要
消毒、灭菌和废弃物管理是安全处理生物因子的关键。因此，了解实验室中可使用的不同消毒、灭菌和废弃物管理方法的基本机制非常重要。具体的清除污染要求取决于所处理的生物因子的性质。本专论描述了被视为生物危害的实验室废弃物的管理和最终处置方法。该信息可用于为特定实验室制定关于清除污染和废弃物管理的标准化和更具体的程序。本专论的目标读者是执行风险评估的人员，例如实验室管理人员或生物安全官员，清除污染实验室物品的实验室人员和科学家，以及处理实验室废弃物的工人。
本专论中关于清除污染和废弃物管理的的信息旨在为世卫组织《实验室生物安全手册》(核心文件) 第四版和其他相关专论提供补充和支持。手册和专论采取了基于风险和证据的生物安全方法，而不是指令性方法，以确保实验室设施、安全设备和工作实践与当地相关、与需求相称且可持续。强调“安全文化”的重要性，该文化包括：风险评估，良好的微生物操作规范和程序，以及标准操作规程，相关的人员入门、进修和指导培训，及时报告事件和事故，随后进行适当的调查和整改。这一新方法旨在促进实验室设计和运行方式，确保更大的可持续性，同时保持对生物安全的充分和适当控制。
其他相关专论提供了详细信息，并助于实施以下专题的系统和策略:风险评估、实验室设计和维护、生物安全柜和其他基本防护装置、个人防护装备、生物安全规划管理、以及暴发疫情准备与应对。
专论中描述了微生物实验室使用的清除污染方法，包括洗手、化学消毒、气体消毒和热力消毒。讨论了不同种类的化学消毒剂及其成分、作用机理和优缺点。还列出了可能影响消毒剂效果的因素。本专论概述了熏蒸方法、热灭活以及如何通过指示剂验证这些处理的有效性。专论还涵盖废弃物管理的各个方面，以及作为废弃物管理组成部分的文档化和记录保存。描述了从实验室中移除液体和固体废弃物及其安全处置的注意事项。还讨论了标本的灭活方法。

本专论支持并借鉴了世界卫生组织《实验室生物安全手册》(1)(核心文件)第四版中关于清除污染和废弃物管理的信息。它包括关于清洁、化学消毒、气体消毒、热力消毒、灭菌、化学和生物指示剂、废弃物管理和生物标本灭活的信息。
其他相关专论提供了详细信息，并帮助于实施以下专题的系统和策略：风险评估(2)、实验室设计和维护(3)、生物安全柜和其他基本防护装置(4)、个人防护装备(5)、生物安全程序管理(6)以及突发事件防范和恢复能力(7)。
在典型实验室中，对可能含有生物因子的废弃物的管理必须包括由风险评估流程确定的经过验证的清除污染手段。清除污染通常采用化学消毒剂和高压灭菌相结合的方法，在某些情况下，辅以焚化。因此，实验室人员对清洁、消毒和灭菌的基本知识的掌握对于实验室的生物安全至关重要。以下一般原则适用于所有已知类别的生物因子。重要的是要确保所使用的任何消毒剂——无论是用于表面还是液体——均被验证为针对实验室中通常处理的生物因子的消毒剂，并且在正确的浓度、正确的温度下使用，并且作用时间至少为已验证的最短接触时间。此外，在选择合适的消毒剂时，一定要考虑有些消毒剂能被有机物灭活；这种有机物可能包括实验室试剂，如液体培养基营养补充剂。
同样重要的是要考虑隔离(洁净材料与受污染材料隔离、人员与受污染材料隔离)和防护(例如，在处理废弃物和将废弃物从实验室工作台转移到高压蒸汽灭菌器期间，确保对废弃物进行安全装袋和装箱)。在第四版《实验室生物安全手册》(1) 3.1小节中，详细描述了良好的微生物操作规范和程序。
在任何繁忙的实验室中，重要的是要有一个专用区域，用于在迅速处理和处置之前短时间内安全存放含有生物因子的废弃物。理想情况下，该区域应与实验室的工作/分析区域分开，并远离储存或处理清洁和无菌物品的区域。有关实验室设计的更多信息，请参阅专论:实验室设计与维护(3)。
本专论中给出的关于清除污染和废弃物管理的通用信息可用于制定标准和更具体的程序，以处理实验室中的病原生物因子。具体的清除污染要求将取决于实验室活动或程序的类型，以及所处理的生物因子的性质。因此，实验室应根据风险评估的结果确定适用于本地需求和条件的适当废弃物管理策略。

# 2.1 清洁和手卫生
# 2.1.1 清洁
一般来说，清洁是指去除不属于物品本身的任何物质。在实验室生物安全范围内的清洁有两个功能:i)它可以清除物品上的污垢和有机物，防止污垢和有机物使化学消毒剂失去活性或阻碍它们与物品内的生物因子接触；ii)清除大部分生物因子，通过随后的化学消毒更有效地将其降低到安全水平。
不应将清洁作为唯一的清除污染过程。化学消毒前的清洁需求是主观的:如果物品看起来在物理上是清洁的，那么就没有必要在化学消毒前对其进行清洁。当某些区域看不到时就会出现困难，例如在不透明或隐藏的管道内。在这类情况下，应进行消毒前清洁的风险评估。
第2节 清除污染方法
如有可能，可用温水通过擦洗手动进行清洁。清洁方法包括刷洗、吸尘、干法除尘、洗涤或湿拖，最好用温水。使用实验室洗瓶机也是一种清洁方法。添加清洁剂(降低表面张力的表面活性剂、清洁剂)可提高清洁效果。常用清洁剂包括苏打溶液（每100 L热水中含3kg碳酸钠（ $( \mathsf { N a } _ { 2 } \mathsf { C O } _ { 3 } )$ ）、肥皂溶液（每100 L热水中含3 kg肥皂）或市售制剂。进行清洁时，必须确保操作人员的安全，防止疾病传播或污染扩散。因此，必须对人员进行培训，并使用个人防护装备、适当的技术和充分的小心。
# 2.1.2 手卫生
虽然合适的手套为佩戴者提供了高度的保护，但它们并不能提供完全的保护，必须在取下手套后洗手。手套会出现破洞，有些可能太小以致于佩戴者不会注意到。液体污染物将通过毛细作用穿过这些孔，并扩散到佩戴者的皮肤上。即使采用了如专论《个人防护装备》(5)中所述的出色的摘手套方法，手在操作过程中也可能受到污染。因此，摘下手套后必须进行手卫生处理。如果没有戴手套，在实验室工作或处理动物后必须洗手。如果在实验室中手被生物因子污染，则污染将出现在皮肤表面。因此，它可以很容易地通过洗手来去除，或者很容易通过抗菌的手消毒液来灭活。
实验室手卫生有两种类型。
# 洗手
用肥皂和自来水进行短时间（约20秒）但彻底的洗手可有效去除实验室获得的污染物。图2.2显示了一种有效的洗手方法。使用抗菌肥皂没有任何优势，因为洗手的目的是去除生物因子，而不是灭活或破坏它们。应在自来水中洗手，因此应使用可将冷热水混合至舒适温度的水龙头。非手触式开关（红外操作开关、或用脚、膝或肘操作水龙头）更值得推荐。如果需要用手打开和关闭水龙头，应使用干净的纸巾将其关闭。应使用一次性纸巾擦干手，纸巾使用后应妥善丢弃在指定的垃圾箱中。
# 酒精搓手
将浓度为 $60 \%$ 至 $9 5 \%$ 的醇类（乙醇、丙醇或异丙醇）涂抹在手上并摩擦至干燥，可有效去除实验室工作中获得的微生物污染(8 9)。使用醇类搓手的正确方法可在世卫组织网站(10)上找到。醇类对于蛋白质或含有蛋白质的物质的穿透力很弱，因此应仅在明显清洁的手上使用。醇类对孢子无活性，对无包膜病毒活性差；如果手可能被这些生物因子污染，应该洗手而不是用含醇的手消毒剂搓手。
# 2.2 化学消毒
生物因子会对化学产品表现出不同的耐药性或敏感性，这取决于它们是否存在细胞壁骨架、脂质膜、多糖层和肽聚糖层。通常，灭活孢子需要比灭活包膜病毒更高浓度的高活性消毒剂。通常，含有1000 ppm（百万分之一）有效氯的次氯酸钠(NaOCl)溶液适用于一般表面消毒，但当处理严重污染、存在有机物或耐消毒剂的生物因子时，建议使用更强的溶液（例如5000 ppm或10 000 ppm）。

# 含氯化合物
次氯酸盐是基于氯的消毒剂，但活性成分是与氯松散结合的氧；这种氧容易损失而可用于氧化其它化合物。这些消毒剂是含有各种平衡组分的溶液，但最活跃的化学物质通常是次氯酸钠(NaOCl)、次氯酸离子(OCl–)和次氯酸(HOCl)。次氯酸盐溶液的氧化能力通常用有效氯的百分比或有效氯的百万分之一(ppm av Cl)表示。(这是历史上的用词不当，实际上是指与氯结合的氧)。
次氯酸盐可被有机物灭活。即使在明显的低污染水平(“洁净条件”)下，也需要较高的浓度，所需浓度随着有机物污染量的增加而增加(“肮脏条件”)；见表2.2。
次氯酸盐溶液可由多种不同的起始试剂制备，如液体漂白剂。既便宜又很容易买到。然而，这些溶液的有效氯含量可能存在不确定性，因为液体次氯酸盐在储存时会衰变；衰变的速度取决于储存条件，主要是温度。稀释的次氯酸盐溶液的保存期有限，约为一天，具体取决于暴露在高温和/或阳光下的情况。
次氯酸钙 $\scriptstyle { \left( \mathrm { C a } ( \mathrm { C l O } ) _ { 2 } \right) }$ 的颗粒或片剂通常含有约 $70 \%$ 的有效氯。因此，含有1.4 g/L次氯酸钙溶液的片剂或颗粒剂溶液将含有约1 g/L有效氯，相当于1000 ppm有效氯。
二氯异氰尿酸钠(NaDCC)是一种固体，即使在高温干燥条件也非常稳定。它含有60%的有效氯，并与活性化学物质次氯酸盐形成平衡。一旦溶解，它与其他形式的次氯酸盐溶液一样不稳定。二氯异氰尿酸钠有多种片剂形式，当溶于规定体积的水中时，可提供特定的有效氯浓度。
使用含有约 $2 5 \%$ 有效氯的氯胺也可进行类似计算。氯胺被认为比其他次氯酸盐来源更能抵抗有机物的灭活(11)。
次氯酸盐可导致金属严重腐蚀，并刺激暴露的皮肤和粘膜。应避免在金属上使用它们，除非制造商声明其设备中的金属与所用次氯酸盐的浓度兼容，或者如果发生腐蚀不成问题，例如，在处置之前正在对物品进行消毒。

二氧化氯 $\left( \mathrm { C l O } _ { 2 } \right)$ 杀微生物的机理与次氯酸盐相似，但更有效。因为它更有效，所以可以在较低浓度下使用，比次氯酸盐腐蚀性小。由于二氧化氯的使用浓度较低，因此与次氯酸盐溶液相比，它往往比次氯酸盐更容易被有机物灭活。
二氧化氯溶液是在使用前将两种成份混合而成。一旦在实验室混合各组分后，溶液的使用时间有限制(具体时间由制造商规定)。
# 酚类
酚类，也称为酚类化合物，是一类由羟基(–OH)直接与芳香烃基团结合而成的化合物。其中最简单的是苯酚 $( \mathsf { C } _ { 6 } \mathsf { H } _ { 5 } \mathsf { O } \mathsf { H } )$ 。苯酚特异性作用于细胞膜，使蛋白质变性，并通过形成不稳定的复合物使胞质内的酶失活。这些相互作用会导致细菌成份渗漏或诱导细胞膜溶解。酚类化合物比次氯酸钠稳定性更高，受溶液中高浓度有机物的影响较小，且能有效对抗细菌繁殖体，尤其是革兰氏阳性菌和包膜病毒。然而，它们对孢子和无包膜病毒无效。尽管有这些优点，但酚类化合物有毒，对皮肤有腐蚀性，并且是已知的呼吸道刺激物。由于这些主要局限性，尽管酚类化合物仍然存在于一些家用消毒剂和清洁产品中，近年来酚类化合物的使用有所减少，它们通常与季铵盐和醇类结合使用。
基于酚类的消毒剂仍被用作消灭分枝杆菌的杀结核菌药剂。原因是这些消毒剂是亲脂性分子，可被分枝杆菌细胞膜上的许多磷脂有效捕获，从而消灭细菌。
# 过氧化物
过氧化物是广泛使用的杀菌剂，因为含有高活性羟基而具有强氧化活性。它们使生物因子的蛋白质和脂质变性，导致膜溶解。当细胞被吸水的氢离子饱和时，可能引起生物因子的细胞发生溶胀。
过氧化氢（ $( \mathsf { H } _ { 2 } \mathsf { O } _ { 2 } )$ ）作为氧化剂，通过产生羟基自由基攻击细胞的基本成分，如脂质、蛋白质和DNA。尽管过氧化氢对细菌孢子和分枝杆菌的活性不尽相同，它具有广泛的杀菌、杀病毒和杀真菌活性。然而，当使用低浓度过氧化氢时，细菌产生过氧化氢酶的能力可以增强对过氧化氢的耐受性(16)。过氧化氢被认为是环境友好型的，因为它可以快速降解为无害的产物——水和氧气。但是，它对皮肤、眼睛和呼吸系统有严重的刺激性。
过氧乙酸（ $( \mathsf { C } _ { 2 } \mathsf { H } _ { 4 } \mathsf { O } _ { 3 } )$ ）是由乙酸、过氧化氢和强酸催化剂混合而成。它是一种比过氧化氢更强的消毒剂。它使蛋白质变性，破坏细胞壁的渗透性，氧化蛋白质、酶和其他代谢产物中的巯基和硫键。
因此，过氧乙酸在低浓度 ${ \mathit { \Omega } } _ { < 0 . 3 \% } ^ { \prime }$ )下具有很强的杀孢子、杀菌、杀病毒和杀真菌作用。过氧乙酸也分解为安全的副产物(乙酸和氧气)，并且具有过氧化氢不具备的不被过氧化物酶分解的附加优点，并且在存在有机物负载时保持活性(17)。它易从化学品供应商处获得，相对便宜，并且可以在使用时进行稀释。但是，与过氧化氢一样，过氧乙酸对皮肤、眼睛和呼吸系统有严重的刺激性。

# 醛
甲醛（ $( \mathsf { C H } _ { 2 } \mathsf { O } )$ ）以福尔马林(约 $3 7 \%$ 甲醛的稳定溶液)或多聚甲醛(一种加热并与空气反应形成甲醛气体的固体聚合物)的形式提供。甲醛会刺激皮肤、眼睛、鼻子和喉咙。高水平暴露可能导致癌症(18 19)；正因为如此，一些国家正在考虑限制其未来的使用。在实验室使用不应使人们接触甲醛或其蒸汽。甲醛只应用于需要它的特定过程，如组织固定，而不应用于一般消毒。不应用于擦拭表面或设备。只有在蒸汽能够完全防护、熏蒸结束后能正确灭活和安全排出的情况下，甲醛才应用于熏蒸空间(参见2.3小节“气体熏蒸”)。
戊二醛（ $( \mathsf { C } _ { 5 } \mathsf { H } _ { 8 } \mathsf { O } _ { 2 } )$ ）在使用前通常需要缓冲(活化)至碱性pH值。活化前，戊二醛是一种稳定的溶液，但活化后，保质期有限。戊二醛的优点是杀菌谱广，腐蚀性低。戊二醛的缺点是有毒性，并且有化学致敏的作用(接触后会引起过敏反应)。使用时必须穿戴适当的个人防护装备，以限制液体或蒸汽形式的潜在暴露。
# 季铵化合物
季铵化合物和类似的化合物，如三胺，是一个多种多样的分子家族，其中一些可用作消毒剂。它们通过表面活性剂活性破坏生物因子的结构来发挥作用。季铵化合物比次氯酸盐更稳定，对呼吸道的刺激性更小，毒性也比酚类化合物低。但是，它们会受到高含量有机物质和具有大表面积且与消毒剂结合的物质的影响，如织物。
它们对非孢子形式的细菌和包膜(含脂质)病毒具有活性。与次氯酸盐或酚类化合物相比，它们对较小范围的生物因子有效，对无包膜病毒、大多数分枝杆菌物种和孢子的效力有限。但是，对主要含有包膜病毒的溶液，季铵化合物可能是最佳的消毒剂。
它们无腐蚀性，但比次氯酸钠贵。如果考虑将其再实验室使用，则在风险评估中需同时考虑潜在的灭活作用和期望其针对的生物因子范围。
# 醇类
用于实验室消毒的醇类有乙醇(通常通过添加甲基化酒精而变性，因此不适合饮用)、丙醇(丙-1-醇)或异丙醇(丙-2-醇)。正常使用浓度为 $70 \%$ ，但根据所用的醇的不同， $60 \%$ 至90%之间的任何浓度都有效(20)。三种醇的活性大致相似；它们有效地对抗广泛的非孢子形式的细菌和包膜(含脂质)病毒。它们对无包膜病毒的活性不同，对细菌孢子无活性。虽然醇类不会被有机物灭活，但在蛋白质存在的情况下，它们的活性是不可靠的；它们可以凝结蛋白质，形成屏障阻止它们进一步渗透到内层。酒精蒸发迅速，便于用作表面消毒剂。然而，它们的快速蒸发也会缩短接触时间，从而降低有效性。

# 2.3 气体消毒
在少数情况下，主要是在采取最大限度保护措施的实验室，风险评估确定需要气体消毒剂(图2.4)，也称为熏蒸剂，对实验室空间、家具和/或设备进行消毒。
如果实验室在难以接近的区域存在广泛污染、设备需要从污染区域运出、或在维护前进行消毒，则可能需要使用气态消毒剂。房间和设备可通过加热多聚甲醛或煮沸福尔马林溶液产生的甲醛气体进行熏蒸消毒。

气体消毒是一个危险的过程，需要经过专门培训的人员操作，当风险评估得出表面消毒不切实际或效率低下的结论时，熏蒸应作为最后的手段。理想情况下，用气体熏蒸消毒的房间应是可密封的，至少在产生甲醛气体之前，房间中的所有开口(窗户、门)应使用不透气胶带密封。熏蒸应在环境温度不低于 $\mathsf { \Omega } ^ { 2 0 ^ { \circ } \mathrm { C } }$ 、相对湿度大于 $70 \%$ 的条件下进行。
无论采用何种熏蒸方法，熏蒸后必须经适当通风人员方可进入。在大多数情况下，工作场所气态消毒剂的暴露程度与检出的限值较低。因此，在紧急情况下，任何人在未通风之前进入熏蒸的房间，都必须穿戴适当的经过测试的呼吸防护装备，并配有正确的过滤器配件。有关呼吸防护装备的更多信息，请参阅专论:个人防护装备(5)。
在室内经过一段时间通风后，在重新进入室内前应尽可能使用经过校准的手持式监控器或内置设备检查熏蒸剂水平。在释放熏蒸剂之前，通常可用碳酸氢铵升华产生的铵中和甲醛熏蒸剂。但是，此步骤不能保证彻底清除熏蒸剂，也不能排除重新进入未充分通风的房间时佩戴呼吸防护装备的必要性；空气中可能仍存在有毒残留化学品。甲醛熏蒸后，房间的墙壁和天花板可能需要用氨水擦拭以中和残留的多聚甲醛。
也可使用过氧化氢或二氧化氯的商业熏蒸系统进行熏蒸。这些系统在安全性、环境保护、熏蒸过程的可控性和速度等方面比甲醛有优势，但成本更高。关于商业系统的使用和有效性的信息可从科学文献获得。

第2节 清除污染方法
# 2.3.1 生物安全柜的气体消毒
在需要维护生物安全柜时，可能需要进行气体消毒(21)。这些情况应在风险评估中明确规定，而不应定期进行生物安全柜气体消毒。目前有可独立产生和循环气态甲醛或过氧化氢蒸汽的设备，用于生物安全柜消毒。这些程序只能由经过适当培训的人员执行。将空气再循环进入实验室的生物安全柜不得使用气体消毒。应使用经验证的表面消毒剂擦拭表面，对生物安全柜进行更频繁的消毒。
# 2.4 热力消毒
加热是生物因子清除污染中最常用的物理方法。干热和湿热都可使用(图2.5)。生物因子与水的接触是蒸汽灭菌的关键，即蒸汽必须到达所有待灭菌或待消毒的物体或材料表面。在高压灭菌中湿热灭菌最有效。在维护良好的、具有经验证的可编程运行程序和互锁门的高压蒸汽灭菌器中，对预清洁物品进行高压灭菌是固体废弃物消毒的金标准。然而，对于小物品和负载，也可以使用高压锅。煮沸不一定能去除所有生物因子，但当其他方法(化学消毒、清除污染、及高压灭菌)不适用或不可用时，可作为消毒的最低限度的处理。

虽然干热也可用于灭菌，但需要更高的温度和更长的时间。例如，在完全无腐蚀性的烘箱中，可用于处理许多可耐受160°C或更高温度2-4小时的物品。燃烧或焚化(参见2.4.2小节“焚化”)也是干热消毒的一种。
# 2.4.1 高压灭菌
压力饱和蒸汽(高压灭菌)是消毒/灭菌实验室材料和废弃物的最有效和最可靠的方法。这是因为，一旦被放入高压蒸汽灭菌器中，在系统完成运行且舱内压力和温度恢复到安全水平之前，无法取出处理物品。
高压蒸汽灭菌器需要能够处理广泛材料和不同类型的负载，这通常需要不同的运行程序(例如，多孔负载、液体、包裹材料)。固体和液体材料消毒有不同的运行程序可供选择。在高压蒸汽灭菌器风险评估过程中，应根据用户定义的标准，如预期用途、废弃物类型和数量，选择高压蒸汽灭菌器。

清除污染和废弃物管理
为了使蒸汽有效地进行消毒，必须排出被高压灭菌材料内的所有滞留空气，并提供足够的消毒时间和足够的蒸汽渗透手段。如果满足这些条件，那么高压灭菌是最可靠的清除污染方法。排出空气的方法有被动和主动两种。
被动：蒸汽从灭菌器舱室的底部产生，或者从舱室顶部进入，空气被迫排出舱室。这是一种更简单、更便宜的方法，但它仅适用于空气排出不受织物或玻璃器皿阻碍的负载。
主动：舱室通过连续的压力变化从舱室中抽出空气。这对于织物、玻璃器皿和其他设备负载是必需的，因为这些负载中滞留的空气无法通过被动方法可靠地排出。空气越难排出，需要的脉冲就越多。该方法是实验室废弃物清除污染的首选方法。

# 高压蒸汽灭菌器运行程序

# 安全预防措施
使用高压蒸汽灭菌器时，必须采取以下安全注意事项。
n 高压蒸汽灭菌器的操作和维护必须安排给经过培训的合格人员。
n 必须提供高压蒸汽灭菌器的操作说明。必须提供包含应用范围(例如固体、液体)和需要维持的参数条件(温度、压力、时间)的灭菌方案。
n 还应提供装载计划(标明可灭菌物品/材料的内容物、数量、体积和质量等信息)。
n 必须制定维护程序，包括必须由有资质的人员对灭菌器舱室、门密封件、仪表和控制装置进行定期检查。
n 必须制定定期验证程序，以确保高压蒸汽灭菌器按设计运行，包括定期使用生物指示剂，并对预真空高压蒸汽灭菌器进行鲍伊-迪克测试和真空泄漏测试，以确认此类高压蒸汽灭菌器中空气的正确排出(参见2.6.2小节“化学指示剂”）
n 必须使用可靠的蒸汽源来提供适当的饱和蒸汽。蒸汽供应必须清洁，以确保材料在使用后无菌，并且不含可能抑制高压蒸汽灭菌器功能或可能损坏高压蒸汽灭菌器管道或腔室的化学物质。
n 放置在高压蒸汽灭菌器中的材料必须放在便于排出空气和具有良好蒸汽渗透性的容器中。
n 高压蒸汽灭菌器腔室里的材料必须松散堆放，以便蒸汽能够均匀渗透。
n 切勿在高压蒸汽灭菌器中处理危险化学品(例如漂白剂、汞或放射性物质)。
n 操作人员必须穿戴适当的个人防护装置，包括在打开高压蒸汽灭菌器时提供热保护的适当的手套、防护服以及眼睛和面部保护，即使温度已经下降到认为打开高压蒸汽灭菌器门是安全的水平也需要穿戴。
n 应小心确保减压阀和高压蒸汽灭菌器的排水管不会被纸、塑料、废弃物或待消毒材料中包含的其他材料堵塞。

# 2.4.2 焚化
焚化可用于处理动物尸体以及解剖废弃物和其他实验室废弃物，可事先进行清除污染，也可不进行清除污染(参见第2节消毒方法)。只有当废弃物运输到焚化炉的过程以受控方式进行，由经培训的人员用合适的运输容器依据运输容器装载程序SOP操作时，才可用焚化替代高压灭菌。移动式和可运输的焚化炉可用于应急和临时设施。必须遵守所有关于焚化的国家和环境立法。
有效的焚化需要有效的温度控制手段和确保所有易燃材料完全燃烧的方式。许多焚化炉和其他焚化废弃物的方法，特别是那些只有一个燃烧室的方法，可能不适合处理感染性材料、动物尸体和塑料。这类材料可能不会被完全破坏，而且烟囱的排放物可能会对大气造成生物因子、有毒化学品和烟雾污染(22)。对于带有二级燃烧室的焚化炉，一级然烧室的温度应至少为 $8 0 0 ^ { \circ } \mathrm { C }$ ，二级燃烧室的温度应至少为 $0 0 0 ^ { \circ } \mathrm { C }$ 。
待焚化的材料即使事先进行了消毒，也应在防漏容器中运输至焚化炉。焚化炉工作人员应得到关于装载、手动搬运和温度控制的正确说明。还应注意的是，焚化炉的有效运行在很大程度上取决于所处理废弃物中的材料类型(例如，有机材料、塑料和纸/纸板)。如果废弃物在可重复使用的容器中运输，还需要考虑运输容器的消毒。
人们一直担心现有或拟议的焚化炉可能对环境产生负面影响，并继续努力使焚化炉更加环保和节能。
# 焚化坑和熔炉
焚化坑是一种传统的使用粘土、水泥或混凝土密封的浅坑露天焚化的方式，待焚化物品被彻底燃烧成灰烬。熔炉的工作方式相似，可能是燃烧一系列废弃物的有效手段。这些方法可能会使坑中的操作人员暴露在有害的燃烧产物和极高的温度下。
# 焚化产物（灰烬）处置
焚化过程可能会浓缩潜在的危险化学品(例如，尸体中的有毒金属和磷酸盐)，焚化灰烬的处理必须符合国家/地方法规。高压废弃物可通过场外焚化或在获得许可的填埋场进行处理。
# 2.5 灭菌
当需要完全消除任何生物因子(包括孢子和朊病毒)时，必须使用灭菌法；例如，对于医疗物品和废弃物，如果风险评估表明需要进行非常严格的清除污染程序时必须使用灭菌法。
灭菌可以通过高压灭菌、某些化学消毒剂、气体消毒、辐照等几种方法来实现。尽管所选方法很重要，但验证和遵守SOP以确保最有效的清除污染也同样重要。为了监测灭菌过程的有效性，需要生物指示剂(表2.5)。

# 2.6 生物和化学指示剂
指示剂通常用于检查和/或监测清除污染过程(清洁、消毒或灭菌)的有效性。它们包括化学、生物，有时还有机械/物理指示剂。
# 2.6.1 生物指示剂
生物指示剂由标准化微生物群组成，它们对特定灭菌过程具有一定的抵抗力(表2.5)。非致病性细菌内生孢子通常用作测试生物，因为它们对灭菌过程具有高度抵抗力，并且在培养时易于检测。如果孢子未被灭菌过程清除污染，它们将萌发并生长，最终释放出二吡啶甲酸，可通过生长培养基中pH指示染料检测到。随着培养时间的延长，培养基的浊度也将表明细菌的生长。不同的内生孢子形成物种会对通常的灭菌过程表现出不同的抗性。嗜热脂肪地芽孢杆菌用于测试熏蒸或高压灭菌处理的效果，而在验证干热或辐照的完成度时，首选枯草芽孢杆菌、萎缩芽孢杆菌或短小芽孢杆菌(表2.5)。

# 3.1.1 污染废弃物的现场处理:分选和储存
包括废弃物在内的受污染和已清除污染材料应清晰可辨，贴上标签(通过使用不同的颜色编码系统或使用生物危害标识)，并分开储存或处置。
分选和储存时需要考虑以下因素：
n 废弃物的性质，例如液体、固体、一般废弃物、感染性废弃物、化学废弃物、利器(受污染或未受污染)和易腐物品；
n 待处理的废弃物量；
n 进行废弃物清除污染的地方(例如，实验室本身、现场、场外)；
n 在清除污染处理之前储存废弃物的潜在必要性；
n 用于储存废弃物的包装的种类和类型(例如，袋子、盒子、箱子、罐子、桶；防液体、防穿刺、耐热、耐化学品、可密封)；

n 内部转移材料和场外运输材料的限制条件；
n 在场外运输前进入受限储存区；
n 储存期限和条件(例如短期或长期储存、温度控制和通风要求)；
n 定期清洁和消毒储存区域的可能性。
离开实验室的生物危害袋应以安全的方式运输和储存，通过使用二级容器、手推车或其他方式，以防止污染储存场所地板和墙壁。适当的包装方法可确保从实验室到清除污染现场所有人员的安全，即使在运输过程中发生损坏也不例外。例如，锐器应收集在防穿刺且不渗透的容器中，容器在封闭后难以打开。急救包必不可少，并且应易于取用。根据温度和废弃物的储存时间，在处理或处置之前可能需要冷却。
所有处理废弃物的人员必须清楚地看到，经过清除污染处理的物品(例如，箱子、袋子、容器)不再存在感染风险。例如，应划掉、移除或隐藏生物危害标志，高压灭菌袋可放入另一个不透明袋中。通常，清除污染后的废弃物被视为城市废弃物，并按城市废弃物进行运输和处置。但是，锐器和其他特殊废弃物(化学、放射性废弃物)必须作为特殊废弃物放置在指定的废弃物容器中。

# 3.1.2 污染废弃物的场外处理：处理和运输
在许多实验室中，感染性废弃物无法在其产生的地方清除污染。因此，必须将其收集并运送至场外设施进行处理和最终处置。当异常数量的受污染废弃物累积时(例如，现场清除污染系统发生重大技术故障或疾病爆发后)，或发现意外污染源且材料需要在最终处置前要对材料进行清除污染时，也可能需要场外处理。
理想情况下，废弃物应包装在联合国认证的容器中，并由持牌承包商运输。应检查承包商对现行法规和指南的遵守情况(作为风险评估过程的一部分)。运输的受污染废弃物必须有以下信息:
n 废弃物类别、废弃物类型，
n 废弃物产生者(例如研究所、实验室)，
n 提货日期，
n 目的地，
n 驾驶员姓名和执照，
n 容器数量和估计体积，
n 从提货区负责人获得的装载收据。

n 清除污染验证记录，
n 场外运输记录和最终处置记录，
n 相关材料安全数据表的数据库(纸质或电子)，
n 应急计划，包括泄露管理应急程序。

# 3.1.4 立法、监管和政策考虑
实验室内产生的含有致病性生物因子的生物危害废弃物的清除污染可在实验室内进行(例如，用漂白剂处理细菌培养物，高压灭菌)，在实验室外的指定清除污染区域进行(例如，机构的集中高压灭菌区域)，或由第三方服务机构在场外进行生物危害废弃物管理或处置(例如，焚化、蒸汽灭菌)。产生废弃物的个人或实验室有责任确保在将废弃物释放或移出实验室之前对其进行安全有效的清除污染，或确保将其安全运送至场外进行清除污染。
对生物危害的实验室废弃物如人体血液和体液一旦得到有效清除污染，就不再被视为对生物有害。但是，如果它们含有生物因子以外的危险(例如化学品、利器、放射性物质)，则仍可视为危险废弃物。有适用于锐器废弃物的国际标准(例如，ISO 23907-1锐器伤害保护)，这些标准应反映在废弃物管理计划中(例如，各种材料的分选程序)。此外，虽然已有效清除污染的人体解剖废弃物不再是生物危害，但社会文化、宗教和审美规范及实践通常会影响对其处置的监管。
国际、国家、省级或市级当局和组织可能会规定其他废弃物管理和废弃物处理注意事项或要求，在制定和实施废弃物管理计划时，应咨询这些事项或要求(23 24)。管辖特定辖区内废弃物清除污染和处置的法律、法规、政策和指南可能会反映出地区差异、当地能力和社会经济条件的差异。经验丰富的人员之间共享的指南和手册、专业行为准则和建议也可通过专业组织或机构获得，以补充强制性要求。

n 生物危害废弃物的定义，
n 废弃物的类别或种类，
n 生物危害废弃物生产者安全处理和处置废弃物的法律义务，

# 国际注意事项
世卫组织的《卫生保健活动废弃物的安全管理：实用指南》第二版，提供可在废弃物管理计划中考虑的危险废弃物处理建议(25)。
某些废弃物清除污染方法也有国际标准。例如，虽然没有关于焚化炉烟囱排放(烟气)的微生物标准，但焚化炉必须符合国际环境要求，例如《关于持久性有机污染物的斯德哥尔摩公约》(26)。对于销毁和清除过程的效率以及可能释放到大气中的物质的浓度，有严格的国际要求，有时也有国内要求。必须证明用于危险废弃物的焚化炉符合这些要求。如果使用第三方生物危害废弃物管理或处置服务，产生废弃物和外包处置的实验室可能会要求它们证明其过程如何符合国际法规。
# 国内注意事项
除国际要求外，废弃物管理方案还必须考虑任何适用的当地法律、法规、政策和指南。在一些国家，当地管辖区对废弃物储存有特定的场地要求，以避免社区暴露在危险废弃物中。例如，为了防止拾荒者和害兽进入，地方当局可能要求将装有生物危害废弃物的容器储存在安全处，直到运输到场外处置。
不同地区对生物危害废弃物的规定可能会有很大不同。例如，在一些国家，即使生物废弃物已被彻底和有效地清除污染(例如，通过消毒或高压灭菌)，也不允许进入一般废弃物处置系统。在这种情况下，废弃物可能必须经过一个清除污染过程，以处理废弃物中剩余的危险物质(例如，化学品、放射性物质、利器、解剖废弃物)。
尊重反映区域差异、当地能力差异和社会经济条件的地方指南也很重要。例如，废弃物管理方案应考虑当地气候是否需要额外的废弃物储存程序(将废弃物储存在阴凉处；如果存在洪水风险，将其存放在高架平台上；或在废弃物产生时立即进行清除污染以避免储存需求)，这一点非常重要。
# 废弃物运输
当废弃物在场外清除污染时，必须按照国家和国际法规进行包装、标签和运输。无论是在一个区域内运输还是出口到另一个国家进行清除污染或处置，生物危害废弃物的运输都受国内和国际要求的管辖，这些要求通常基于《联合国关于危险货物运输的建议:示范条例》(第27条)。
如果没有国家法规，负责的市政当局可以参考联合国的建议。空运生物危害废弃物必须遵守国际民用航空组织的《危险物品安全航空运输技术细则》和国际航空运输协会(IATA)的《IATA危险品规则》，这两份文件均基于联合国的建议(2829)。
除了国际公约和协定之外，危险废弃物跨越国际边界的移动可能受到进口国、出口国或两者的监管(30-32)。

# 3.2 液体废弃物的清除污染
液体废弃物的处理是研究设施面临的一个重要问题。有多种方法可供选择，所有这些方法的优缺点都需要进行风险评估。方法的选择取决于废弃物的化学成分、待清除污染的生物因子以及每种方法的初始和持续成本。液体废弃物的清除污染方法在以下各小节概述。
# 3.2.1 污水管道系统
根据风险评估，对于附属于技术先进的市政下水道系统的研究机构或临床设施中操作的低风险样本，将液体废弃物容器直接排空至下水道可能是一种适当的处置方法。虽然在技术上不是一个清除污染过程，但附属于废弃物处理厂的下水道系统将在液体废弃物排入环境之前对其进行充分消毒。对于需要加强控制措施或最大限度遏制措施(由风险评估确定)的生物因子或实验室程序，不允许排放到市政下水道。事实上，即使对于危险性较小的生物因子，当地或国家法规也可能不允许使用这种方法。将液体生物废弃物混入化学物质(如乙醇、甲醛或盐酸胍)再排放入污水系统中可能也是被禁止的。在任何情况下，都需要严格遵守当地和国家法规。
# 3.2.2 化学消毒
大多数研究实验室禁止未经任何处理就直接处置生物因子；因此，化学消毒已经成为在处置之前清除污染溶液的标准手段。风险评估有助于确定液体废弃物最合适的化学消毒方法。
选择化学消毒剂时，首先要考虑化学消毒剂对实验室所处理的生物因子的活性。一般而言，研究中使用的是预先划定的生物因子，但在临床标本中可发现范围更广的生物因子。孢子和朊病毒在处置前需要更严格的清除污染过程。同时，必须考虑有机负荷(与生物因子混合的有机物量)，因为大多数化学消毒剂(包括次氯酸盐)都能被有机物灭活。
考虑即用型消毒剂的稳定性很重要，因为它会影响更换消毒剂的频率。还应考虑消毒剂及其刺激物的毒性和腐蚀性。此外，出于可持续性原因，可能需还要考虑成本和保质期。
用于液体废弃物清除污染的最常见化学品是次氯酸钠、苯酚和季铵化合物。
在对任何溶液进行化学消毒之前，必须考虑液体中的其他化学品，以避免混合不相容的化学品产生不良影响。硫氰酸胍是一种常见的离液剂(能够破坏氢键)，用于在核酸分离和测序或聚合酶链式反应之前使细胞变性。将含有硫氰酸胍和次氯酸钠的溶液混合，会产生盐酸和氰化氢的有毒气体混合物。甲醛溶液和次氯酸钠的组合会产生多种有毒气体，包括盐酸、氯气和甲酸。乙醇与含次氯酸钠的溶液混合生成氯仿。
# 3.2.3 高压灭菌
高压灭菌液体废弃物可能是一种优选的灭菌方法，因为处理后的液体通常可以通过下水道系统进行处理，而无需担心化学消毒剂或其副产品污染环境。如2.4.1小节中所述。高压灭菌时，蒸汽必须与待消毒溶液直接接触，因此盛放液体废弃物的容器不应密封。废弃物容器应位于二级安全容器内，但该容器不应高于废弃物容器，以便用蒸汽置换空气。可以使用重力置换或预真空程序，但高压蒸汽灭菌器应具有特定的液体程序，在程序完成后缓

# 3.3 固体废弃物清除污染
对于具有潜在传染性的固体废弃物的处理，化学消毒很少是首选方案，因为很难确保化学品与所有表面接触。对于大多数固体，高压灭菌或焚化是首选方法。对于大型动物尸体，则进行焚化或碱水解，并进行炼制(参见“炼制”）。结合加热、粉碎或混合，和/或碱性消化的商业化清除污染系统市场有售。
# 3.3.1 高压灭菌
高压灭菌固体废弃物，如2.4.1小节“高压灭菌”所述，是对固体废物中的生物因子进行灭菌的可靠方法。然而，一个关键问题是需要确保废物的所有部分都暴露在蒸汽中；被滞留的空气起到了隔离作用，并可能使部分废物保持潜在的传染性。如果废物包装不规则，且固体废物中含有液体，则更难确保这种接触。完全干燥的物品，如手套和工作服，可能会困住干燥空气形成气穴，无法实现消毒。因此，放置在高压蒸汽灭菌器中的废物或材料必须放在易于排出空气和良好的蒸汽/热渗透的容器中。必须通过使用干废物测试样和在该试验废物中放置生物指示剂来验证运行条件。必须通过实验确定固体废物中对额外水的需求；一些研究表明，在特定条件下，添加额外的水可能有助于消毒(33)。为了使整个固体废物达到消毒温度，可能需要比通常用于消毒材料更长时间的消毒周期。必须避免大而笨重的材料、大型动物尸体、密封的耐热容器和其他阻碍热量传递的废物。
通过高压蒸汽灭菌对动物尸体清除污染是一项特殊的挑战。尸体可冷冻储存直至收集到足够数量再进行清除污染。无论采用何种尸体解冻方法，在任何运行程序或运行时间中，都必须增加完全解冻所有材料所需的额外时间。
此外，必须提前并定期进行实验验证，以证明所有尸体均达到适当的温度(通常为$1 2 1 ^ { \circ } \mathrm { C }$ (34)。持续的清除污染验证是必要的。可以通过以下两种方式验证成功的清除污染:将生物指示剂置于散布在废弃物容器中的巧妙放置的废弃物测试样本中；或者通过使用连接到杆上的指示剂，这些指示剂可以插入和收回而不干扰废弃物容器的内容物。

# 3.3.2 焚化
焚化是一种消除固体废弃物中所有已知生物因子的方法，包括孢子和朊病毒；理想情况下，焚化是在技术先进的焚化炉中进行的。如2.4.2小节“焚化”中所述，主燃烧室内的正确温度和燃烧时长对于确保固体废弃物的完全燃烧和任何生物因子的清除污染至关重要。此外，焚化炉操作员需要接受在将废弃物装入焚化炉之前如何安全处理废弃物的培训。如果要使用可重复使用的容器，则需要在开始焚化之前确定这些容器的消毒方法。
除了生物因子之外，焚化固体废弃物时还需要考虑另外两种材料:塑料和钠钙玻璃。实验室使用的大多数塑料比废纸燃烧释放更多热量，如果焚化炉中的塑料量超过焚化炉制造商推荐的量，可能会使焚化炉过热。钠钙玻璃在约 $5 5 0 ^ { \circ } \mathrm { C }$ 下熔化，会覆盖耐火砖并缩短其寿命；因此，最大限度地减少固体废弃物中钠钙玻璃的掺入非常重要。钠钙玻璃焚化产生的灰可能富含重金属和磷酸盐，需要进行特殊处理和处置。
# 3.3.3 碱性消化
碱消化是在1N或1N以上的碱存在的情况下，利用升高的温度和压力，将尸体中的大多数细胞物质分解为可溶形式。该工艺可将尸体分解至可溶性部分和富含钙的固体残留物。碱性消化可有效清除几乎所有已知的生物因子。在150°C下，在1 N氢氧化钠(NaOH)或氢氧化钾(KOH)存在的情况下进行消化，已证明可使朊病毒无传染性(35)。该过程需要大量的能量和时间(几个小时)，应该仅在有大量尸体(每周几公斤)需要处理的机构中使用。
# 3.3.4 炼制
炼制是另一种清除污染动物尸体的方法，在压力容器中使用约 $1 3 0 ^ { \circ } \mathrm { C }$ 的蒸汽将尸体分解为脂肪、蛋白质和骨骼。该方法传统上用于较大的动物，可有效消除大多数生物因子。然而，炼制对朊病毒无效，不应用于感染朊病毒的动物尸体处理。

# 灭活方法
清除污染过程目的是使废弃物能够安全处置，或去除或销毁实验室表面的生物因子。灭活过程是确保任何含有生物因子的材料的安全，而后可以在不太严格控制的实验室中进行处理。通常，被灭活的材料含有高浓度的生物因子，可能需要非常有效的处理方法来确保完全灭活。可能还需要一些过程来保持蛋白质、核酸、其他生化化合物和结构的稳定性，以供将来分析。因此，此类流程需要验证。从具有较高风险控制措施的区域移除传染性材料以供后续处理可能有几个原因。最常见的原因是提取核酸用于核酸扩增，如聚合酶链式反应。对于实验室生物安全，生物因子的灭活降低了后续工作的风险，并使得在大多数情况下能够在核心要求下工作。在核心要求下工作的优势包括:更快的工作流程，不需要对受化学消毒剂影响的设备进行消毒，以及工作空间维护成本更低。根据风险评估，可能需要两种不同的灭活方法来灭活生物因子。
建议在灭活方法中使用对照或生物指示剂。然而，灭活方法的验证通常仅使用实际的生物因子进行，并使用对照来监测其他过程与灭活方法相结合的的效力，例如核酸提取或标本的交叉污染。
# 4.1 标本灭活
标本灭活是一种分析前处理，用于去除/灭活生物因子。还有其他需要灭活样本的原因，如固定和终止反应。因此，灭活方法的选择取决于风险评估和后续实验步骤。
在风险评估中，需要考虑从样本种分离的液体介质，因为液体介质的清除污染方法通常不同于灭活样本的方法。
如果消毒剂用于清除污染目的(泄露)，则实验中的阳性对照可确保标本不与消毒剂接触。对于大多数灭活程序，未对生物因子的灭活效果做对照，但必须在灭活程序的初始验证中使用对照，以证明所选方法的有效性。
# 4.1.1 核酸提取
当使用遗传技术时，通常需要进行核酸提取使核酸从细胞或病毒颗粒中释放出来用于扩增。需要高效提取核酸，以便准确估计生物因子的量。然而，出于生物安全和生物安全保障目的，可能需要完全灭活。如果已知生物因子的感染剂量较低，则其灭活将尤为重要。出于聚合酶链式反应的目的，灭活通常使用含有蛋白变性剂(如胍盐)的裂解液或直接加热，或两者兼用。

# 4.1.2 载玻片的热灭活
在临床实验室中，经常通过加热载玻片，在染色前将样本粘合或粘附到玻璃上。对于福尔马林固定、石蜡包埋的组织标本，通过加热清除多余的石蜡。热处理不是灭活过程，生物因子已经通过福尔马林固定而灭活。然而对于可能含有朊病毒的组织切片而言，即使在载玻片加温器或类似加热设备上进一步处理，也只能将其视为潜在污染。
对于许多临床过程，包括对各种细菌进行革兰氏染色和结核分枝杆菌的疑似患者制备痰标本进行泽赫尔-尼尔逊染色时，一个关键步骤是使用载玻片加温器在载玻片上干燥标本。然而，除非在涂片中加入 $5 \%$ 的苯酚，否则热处理无法完全灭活结核分枝杆菌(36)。
尽管热灭活可能会减少载玻片上的生物因子数量，但不应假设加热至干燥足以灭活载玻片上的所有生物因子。只有在下列情况下才能假定所有生物因子均已失活:加热已通过实验验证；已对加热表面进行了评估确保在整个工作表面上温度是均匀的；并且有确认温度的方法。
# 4.1.3 组织的甲醛处理
甲醛广泛用于病理和解剖标本的固定。尽管这种处理主要是为了保存组织的组织学特征，但甲醛也可靠地使生物因子失活(如2.2.1消毒剂的类型和2.3气体消毒中所述)。
甲醛以福尔马林的形式用于组织固定，通常为含 $4 \%$ 甲醛的福尔马林溶液。甲醛以约1mm/小时的速度穿透组织(37)；因此，组织在福尔马林中的固定的时间取决于样品的厚度。通常，甲醛固定过程不设对照，因为这是不切实际的，且甲醛的使用基于SOP的验证。
在估计的孵育时间后，在样本上小心切割，通过观察组织的颜色可以表明在安全处理样本之前是否需要更长的孵育时间。然后将组织加工成福尔马林固定、石蜡包埋的组织标本，用于显微镜检查和分子生物学分析。甲醛会导致蛋白质和核酸的氨基交联，从而导致福尔马林伪像，如变性蛋白质和受损DNA。因此，后期分析受到限制，需要加入一些特殊流程(如基于抗体的检测方法——抗原修复)(38)。
甲醛是一种刺激物和致癌物质(18 19)，其分析和医疗用途(作为组织学的固定剂或对死者进行防腐处理)正在评估中。
# 4.1.4 电离辐射
电离辐射作为一种灭菌剂具有显著优势:这种辐射能够通过封闭包装(不需要与生物因子直接物理接触)使生物因子失活，并且在失活过程中不会留下任何残留物(如化学品)。由于电离辐射很容易穿透所有物质，它能够使任何类型的生物因子失活，包括孢子。然而，它通常很昂贵，需要专门的设施，并且比大多数处理需要更多的时间。
电离辐射的使用限于两种作用：(一)使少量生物因子失活，以用于以后的结构或免疫分析或用作疫苗；或(ii)对体积大的物品/材料进行灭菌，通常是可能因接触蒸汽或化学消毒剂而损坏的医疗用品或设备。
有三种商业电离辐射源:γ辐照器、x光机和电子束加速器，也称为电子束。它们都通过辐射与物质相互作用时产生的自由基使生物因子失活。这些自由基产生核酸链断裂和加合物，这通常被认为是失活的关键损伤。一般而言，辐射敏感性与遗传物质的大小直接相关，D10值的范围从鼠伤寒沙门氏菌的0.2千戈瑞(kGy)到口蹄疫病毒的13.0千戈瑞(39)。

# γ辐照器
γ辐照器通过放射性元素的衰变产生高能光子；钴-60 (60Co)或铯-137 (137Cs)是典型的γ辐射源。钴辐照器的半衰期较短(60Co为5.3年，而137Cs为30年)，因此需要更频繁地补充新鲜钴。然而，钴辐照器可提供更强的γ光子(60Co产生的两种γ射线为1.17和1.33兆电子伏(MeV)光子；而对于137Cs则为660千电子伏(keV)光子)，因此60Co可对更大质量的物质进行清除污染。
必须考虑放射性元素的半衰期，以确保待灭活材料的适当暴露。此外，钴源是金属，对于用于大量灭菌的房间大小的商业辐照器，可以浸没在水池中以提供屏蔽；铯源通常以动力盐的形式提供，因此它们通常不能在水中使用。
γ辐射源被盗是国家当局关注的问题，因为它们可能被用来传播放射性污染。获得γ辐照器变得很困难，因此许多国家正试图用X光机替代。
# X光机
X光机通过加速电子进入金属靶产生电离辐射。电子和金属靶之间产生的相互作用，要么击出轨道电子，要么改变加速电子的方向，导致X光光子的产生。与伽马射线照射器不同，X光机可以关闭，因此本质上是更安全的设备。然而，来自X光机的光子能量通常比γ辐照器低得多。较低的能量可能导致较低的剂量率，这意味着用X光机消毒比γ辐照器需要更长的暴露时间。
鉴于X射线或γ光子的穿透能力，通常需要在设备和被辐射材料周围用混凝土或钢材进行屏蔽。以将潜在的辐射暴露量降低至国家设定的监管限值以下。国家主管部门还可为使用γ辐照器或X光机的人员设定暴露限值，这可能需要使用个人剂量计来跟踪人员的暴露情况，以确保他们不会过度暴露于辐射。
# 电子束机
能量高达10 MeV的电子束机已经上市。与X光机类似，切断电源会使电子束机停止电子的产生，这使得它们比γ辐照器更安全。除了较高的成本之外，电子束的主要缺点是电子在水中或类似密度的材料中的穿透深度相对较短(6 MeV束约为4cm)(40)。电子直接与靶中的物质相互作用，破坏原子键并产生次级电离粒子，从而使关键靶(通常是核酸)失活。此外，高原子序数材料，如金属，必须远离电子束，以防形成高能X光。如果严格遵守只使用玻璃或塑料容器的规定，电子束装置可以独立使用，除了制造商提供的屏蔽外不再需要额外的屏蔽。
对于每种类型的辐照器，必须仔细计算辐射的穿透力(每分钟每厘米材料的辐射剂量)，并为处于常规使用的相同物理状态(例如液体、冷冻、冻干)的生物体绘制存活曲线。对于细菌繁殖体而言，氧状态会显著影响存活率，验证实验中的氧含量必须与常规使用时相似。
当使用辐射作为灭活方法时，充分的剂量测定(评估物体吸收的电离辐射剂量)也是必不可少的。剂量测定可能是一个挑战，因为，例如在每毫升含有106个埃博拉病毒粒子的溶液中，通过辐射使得病毒粒子存活概率为百万分之一的剂量可能在30 kGy数量级(41)，远远高于大多数化学剂量测定计或生物指示剂(如短小芽孢杆菌孢子)的范围。这种情况下必须用丙氨酸自由基剂量测定法(42)或荧光膜剂量计(43)等技术的专用剂量计进行测定。
如果辐照灭活时，如不能保持在确认实验中确定的总体积、氧浓度、待灭活生物因子浓度、材料物理状态(固体、液体)和生物体状态(繁殖体与孢子)的限值内，可能导致生物体在电离辐射程序中存活，并可能释放受污染的材料。
