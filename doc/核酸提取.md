# 核酸提取试剂说明书
【产品名称】通用名称：核酸提取试剂
【包装规格】Y 型：16 次/盒，24 次/盒，48次/盒，96次/盒；
B 型：50 次/盒，100 次/盒，1000 次/盒。
【预期用途】用于血浆、血清样本游离核酸提取。
【检验原理】本试剂盒以超顺磁性的纳米磁性粒子为基质，能够特异性识别核酸分子并高效结合，在独特的缓冲体系和外加磁场作用下，从血清、血浆等样本中分离纯化高质量游离核酸。提取产物可用于实时荧光定量PCR、二代测序等各种下游实验，适合高通量工作站的自动化提取。
【主要组成成份】本试剂盒由裂解结合液、洗涤液、磁珠、洗脱液等组成。

自备仪器：恒温混匀仪、磁力架、离心机等；

# 2. 自动法
自备仪器：24通道核酸提取仪。

将冻存血浆在 $3 7 ^ { \circ } \mathrm { C }$ 解冻后颠倒数次混匀，然后 $1 1 0 0 0 \mathrm { r p m } 4 ^ { \circ } \mathrm { C }$ 离心 $1 0 \mathrm { { m i n } }$ ，取上清液体进行提取。

# 2. 样本提取
2.1 将离心管置于磁力架上静置吸附 $3 { \sim } 5 \ \mathrm { m i n }$ ，待磁珠完全吸附时小心去除液体，取下离心管。
2.2 加入1mL洗涤液，振荡混匀1min，使磁珠充分悬浮，短暂离心以去除内壁的滴液。
2.3 将离心管放置于磁力架上静置吸附1min，待磁珠完全吸附时使用小心去除液体，取下离心管。
2.4 重复操作步骤2.2、2.3一次。
注意：可将步骤 2中所有磁珠及液体转移至 $1 . 5 \mathrm { m L }$ 规格的离心管内再进行后续操作。
2.5 加入 $1 \mathrm { m L } 8 0 \%$ 乙醇 (自备) ，振荡混匀1min，使磁珠充分悬浮，简短离心以去除内壁的滴液。
2.6 将离心管放置于磁力架上静置吸附1min，待磁珠完全吸附时小心去除液体，简短离心以去除内壁的滴液，室温晾干 $5 { \sim } 1 5 \ \mathrm { m i n }$ 。
注意：乙醇残留会抑制后续的酶反应，所以晾干时要确保乙醇挥发干净，但也不要干燥太长时间，以免核酸洗脱不彻底，磁珠不反光为宜。
2.7 将离心管从磁力架上取下，加入 ${ 5 0 } { \sim } 2 0 0 \mu \mathrm { L }$ 洗脱液，振荡混匀使磁珠悬浮。将离心管放于恒温震荡仪上，室温 $1 0 0 0 \mathrm { r p m }$ 振荡洗脱 $5 \mathrm { { m i n } }$ ，短暂离心去除管壁及管盖上液体。
2.8 将离心管放置于磁力架上静置吸附 $2 \mathrm { m i n }$ ，待磁珠完全吸附时小心将DNA溶液转移至1.5mL收集管中，并于适当条件保存。

# 自动化提取（货号：X1751R）
1. 取出预封装深孔板，颠倒混匀数次使磁珠重悬，轻甩孔板使试剂及磁珠均集中到孔板底部（也可使用孔板离心机， $5 0 0 \mathrm { r p m } { \times } 1 \mathrm { m i n }$ 进行离心），使用前小心撕去铝箔封口膜，避免孔板振动，防止液体溅出。
2. X1751R 洗脱板位 5 为空板，需手动加入 $7 0 \mu \mathrm { L }$ 的洗脱液。
3. 将血浆样本平衡到室温，板位1中加入 $2 \mathrm { m L }$ 样本和 $2 0 0 \mu \mathrm { L }$ 蛋白酶K；将板位按顺序放入核酸提取仪器，点击Y107-G51 程序运行。
4. 自动化程序结束后，将洗脱液转移至新的无核酸酶离心管中，进行下一步操作，或者保存于- ${ \cdot 2 0 ^ { \circ } } \mathrm { C }$ 。
# 【检验结果的解释】
1. 如出现磁珠残留，可以用磁力架吸附磁珠，吸取上清。磁珠残留的解释：
1.1 磁珠残留孔几个：a) 磁棒套与板子的不配适，磁棒套变形；b) 仪器不水平或者磁棒变形。
1.2 磁珠残留很多且大面积：a) 仪器不水平或者仪器故障；b) 没有在储存条件及有效期内使用；c)磁棒无磁性。

# 【注意事项】
1. 若溶液存在析出现象（温度过低），请将试剂或预封装板子放置于 $3 7 ^ { \circ } \mathrm { C }$ 烘箱中，直至溶液完全溶解，再进行下一步提取。
2. 样品应避免反复冻融，否则会导致核酸提取量降低。样本可立即进行提取，也可于 $4 \mathrm { { } ^ { \circ } C }$ 保存待测，保存时间不超过 $^ { 2 4 \mathrm { h } }$ 。长期保存可置于- ${ \bf \cdot } 2 0 ^ { \circ } { \bf C }$ 或- $. 5 0 \mathrm { { } ^ { \circ } C }$ 冷冻保存。
3. 本试剂盒提取基因组 DNA，操作过程要特别注意防止核酸降解。所使用的器皿、加样器等均为专用，离心管、枪头等一次性耗材需进行高压灭菌。操作人员应戴无粉手套、口罩等。
4. 使用前请详细阅读使用说明，严格按照使用说明书操作，临床样本等需在生物安全柜中进行。
5. 洗脱时可能存在磁珠残留，吸取样本时应尽量避免吸入磁珠。
6. 妥善处理所用样本及试剂材料，彻底清洗并消毒所用操作台面。
